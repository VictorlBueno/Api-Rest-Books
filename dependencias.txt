! Nodemon
Atualiza automáticamente o NodeJS a cada atualização
- 1. npm install nodemon -D
- 2. Adiciona um script = "dev": "nodemon server.js",
- 3. Inicia o nodemon com o server = npm run dev

! Express
O Express fornece uma camada fina de recursos fundamentais para aplicativos da web, sem obscurecer os recursos do Node
- 1. npm install express || npm install express-generator -g = vem com esqueleto pronto
- 2. no arquivo: import express from "express";

! Eslint
Padroniza o código. Exemplo: padrão de aspas e ponto e vírgula
- 1. instalar com npm init @eslint/config
- 2. aplicar os padrões com npx eslint ./src --fix
- 3. para sempre manter a padronização automática CTRL + SHIFT + P e na última linha colocar:
    "editor.codeActionsOnSave": {
        "source.fixAll.eslint": true
    }
